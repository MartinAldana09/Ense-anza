# Simulador de Transformaciones Geométricas

Aplicación interactiva para visualizar transformaciones geométricas en tiempo real.

## Características

- Selección de figuras
- Traslación, rotación, escala y reflexión
- Visualización dinámica
- Representación matricial
- Descarga de configuraciones

## Ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
