# Transformaciones Geométricas Interactivas

<div align="center">

## Exploración Visual del Plano Cartesiano

Aplicación interactiva desarrollada con Streamlit para aprender,
experimentar y analizar transformaciones geométricas en tiempo real.

---

<table>
<tr>
<td align="center">

### Traslación
Mueve figuras en el plano  
sin alterar su forma.

</td>

<td align="center">

### Rotación
Gira figuras alrededor  
de un punto fijo.

</td>

<td align="center">

### Reflexión
Construye imágenes  
especulares.

</td>

<td align="center">

### Homotecia
Amplía o reduce figuras  
manteniendo proporciones.

</td>
</tr>
</table>

## Vista General

</div>

### Herramientas incluidas

- Creación de figuras personalizadas
- Figuras predefinidas
- Superposición de transformaciones
- Visualización en tiempo real
- Representación matricial
- Centro de rotación configurable
- Reflexiones respecto a ejes
- Escalamiento dinámico
- Descarga de configuraciones
- Evaluaciones interactivas
- Actividades de aprendizaje

---

# Visualización Matemática

La aplicación permite observar:

- cambios de posición
- cambios de orientación
- cambios de tamaño
- composición de transformaciones
- efecto del orden de aplicación

---

# Módulos de la Aplicación

| Módulo | Descripción |
|---|---|
| Aprendizaje | Explicaciones intuitivas y ejercicios |
| Simulador | Experimentación libre de transformaciones |
| Evaluación | Resolución de problemas geométricos |
| Representación matricial | Visualización de matrices y vectores |
